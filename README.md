Student Management System
📌 Project Description

This is a Student Management System developed using Java and Spring Boot. The application allows users to manage student records efficiently with basic CRUD (Create, Read, Update, Delete) operations.

🚀 Features
Add new student
View all students
Update student details
Delete student
Search student (optional)

🛠️ Tech Stack
Java
Spring Boot
MySQL
HTML, CSS
Thymeleaf (if used)

🗄️ Database Setup
Open MySQL Workbench or phpMyAdmin
Create a new database:
CREATE DATABASE student_db;
Import the provided student_db.sql file
Update your application.properties file with your MySQL username and password

⚙️ How to Run the Project
Clone the repository:

git clone https://github.com/mandavkarlrushali-star/student-management
Open the project in STS (Spring Tool Suite)
Configure MySQL in application.properties
Run the project as Spring Boot Application

Open browser and go to:

http://localhost:8080
